This project is a comprehensive tool designed to visualize fundamental sorting algorithms and benchmark their performance. It features both a Java Swing desktop application for deep performance testing and a Web-based interactive visualizer.

Features

  • Dual-Platform Visualization: Run animations via the Java desktop GUI or the web interface (index.html).
  • Real-Time Controls: Start, stop, and adjust animation speed to observe how elements are swapped and compared.
  • Performance Benchmarking: Automated tests across various data sizes (100 to 10,000 elements) to measure execution time.
  • Data Export: Save performance results directly to a results.csv file for analysis.
  • Algorithm-Specific Themes: Unique color coding for each algorithm (e.g., Pink for Insertion, Blue for Merge).

Project Structure

  • Main.java: The entry point where users choose between Visualizer and Performance Test modes.
  • algorithms/: Contains Java implementations for Insertion, Selection, Merge, and Quick Sort.
  • gui/: Handles the desktop interface, including the control panel and visualization rendering.
  • Performance/: Contains the logic to execute benchmarks and handle CSV file I/O.
  • Web Assets: index.html, script.js, and style.css for a standalone browser-based experience.

How to Run

1. Java Desktop Application

  1. Open the project in your preferred Java IDE (IntelliJ, Eclipse, etc.).
  2. Run Main.java.
  3. Select "Visualizer" for animations or "Performance Test" to generate data.

2. Web Visualizer

  1. Locate the index.html file in the project directory.
  2. Open it with any modern web browser to start the interactive simulation.
